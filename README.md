# ⏰ Python Voice Reminder App

A simple Python reminder application that uses text-to-speech to remind users about tasks after a specified time. The program asks the user what they want to be reminded about and after how many seconds, then notifies them using voice and text.

---

## 🚀 Features

- 🔊 Text-to-speech reminder using Python
- ⏱ Set reminder time in seconds
- 💬 Voice and console notifications
- 🧠 Simple and beginner-friendly Python project

---

## 🛠️ Tech Stack

- Python
- pyttsx3 (Text-to-Speech Library)

---

## 📦 Installation

1. Make sure Python is installed on your system.

2. Install the required dependency:

```bash
pip install pyttsx3
```

---

## ▶️ Usage

Run the Python script:

```bash
python reminder.py
```

Then follow the instructions:

1. Enter what you want to be reminded about  
2. Enter reminder time in seconds  
3. The app will notify you using voice and console message

---

## 💻 Example

```
Welcome to your reminder app!
What should I remind you about: Drink water
After how many seconds you want to be reminded: 10

Drink water set for 10 seconds...
Reminder! Drink water
```
